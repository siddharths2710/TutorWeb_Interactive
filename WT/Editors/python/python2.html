Variables & Operators

Python is a scripting language, or a loosely dynamic high-level general purpose language. This means that variables do not need to be declared before using them.Python uses a very clever principle known as “Dynamic Typing” to figure out the data types of the variables.The datatype of the variable gets determined at runtime. This adds as an advantage while programming in Python instead of C,C++ or Java. This however means more processing than efficiency.

Creating web apps, games, and search engines all involve storing and working with different types of data. They do so using variables. A variable stores a piece of data, and gives it a specific name.

For example: spec = 11

The variable spec now stores the number 11.




Operator Types

An operator is a symbol that is used to perform specific mathematical or logical functions.Python is rich in built-in operators and provides the following types of operators

  • Arithmetic Operators

    1. Addition 3 + 4 = 7
    2. Subtraction 4 - 3 = 1
    3. Multiplication 3 * 4 = 12
    4. Division 4/2 = 2
    5. Exponent 3 ** 4 = 81
    6. Floor Division 9//2 = floor(4.5) = 4

  • Logical Operators

    1. == 10==10 is true but 10==20 is false
    2. != 10==20 is false but 10==20 is true
    3. >= 10>=10 is true & 10>=20 is false
    4. <= 10<=10 is true & 10<=20 is true
