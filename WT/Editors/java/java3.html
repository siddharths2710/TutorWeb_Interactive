For loop

As we all know, the for loop is used to execute a sequence of statements multiple times and abbreviates the code that manages the loop variable. There are two types of for loops in java.

//Original for loop
for(init;condition;step)
{
   //Statements
}
            


//Enhanced for loop
for(declaration : expression)
{
   //Statements
}

The enhanced for loop is used to traverse collection of elements including arrays.
