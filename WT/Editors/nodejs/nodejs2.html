Variables in Node JS

Since node.js is a JavaScript-based framework, it is bound to support Javascript rules of declaring variables and operators. All node.js variables must be identified with unique names.These unique names are called identifiers. Identifiers can be short names (like x and y), or more descriptive names (age, sum, totalVolume).
